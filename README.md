# 五子棋 · Gomoku

在线五子棋对战，支持本地双人、在线匹配，以及隐藏 AI 对手。

## 部署

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/WWWyyy452/gomoku-online)

点击上方按钮一键部署到 Render（免费）。

## 本地运行

```bash
pip install -r requirements.txt
python app.py
```
